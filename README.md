# 📊 E-Commerce Sales Analytics Dashboard

An interactive **E-Commerce Sales Analytics Dashboard** built using **Python, Pandas, Plotly, and Streamlit**.
This project analyzes sales performance, customer segments, and product trends using interactive visualizations and also includes a **Machine Learning model to forecast next month’s revenue**.

---

## 🚀 Features

* 📈 Sales Performance Analysis
* 🧑‍🤝‍🧑 Customer Segmentation Insights
* 🛍 Product Trend Analysis
* 📊 Interactive Visualizations using Plotly
* 🤖 Machine Learning based Revenue Forecasting
* ⚡ Real-time Dashboard using Streamlit

---

## 🛠 Tech Stack

* Python

## ▶️ Run Locally

Clone the repository

```
git clone https://github.com/kanha165/ecommerce-sales-analytics-dashboard.git
```

Go to the project directory

```
cd ecommerce-sales-analytics-dashboard
```

Install dependencies

```
pip install -r requirements.txt
```

Run the Streamlit app

```
streamlit run dashboard/app.py
```

---

## 🌐 Live Deployment

This dashboard can be deployed using **Render or Streamlit Cloud**.

Start command:

```
streamlit run dashboard/app.py --server.port $PORT
```
